Spray Rest 项目框架

用如下步骤开始工作

1. Git-clone this repository.



2. Change directory into your project:

        $ cd blabla

3. Launch SBT:

        $ sbt

4. Compile everything and run all tests:

        > test

5. Start the application:

        > re-start

6. Browse to http://localhost:8080/

7. Start the application:

        > re-stop
